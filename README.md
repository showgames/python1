# バウンズゲーム

* ボールを落とさずひたすら壁に当て続ける

----------------------------------------
## 動作環境

* Python 3.4.3 以上

----------------------------------------
## 起動

* python ShellからソースをOPEN
* Run Module

----------------------------------------
## 遊び方

* 画面をクリックするとスタート
* 左右方向キーを使ってボールを跳ね返す
* レベルが上がってくるとスピードが変化する
